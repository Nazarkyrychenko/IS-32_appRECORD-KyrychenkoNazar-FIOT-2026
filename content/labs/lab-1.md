## Тема, Мета, Місце розташування

**Тема:** Вибір предметної області. Аналіз, моделювання та розроблення адаптивного WEB-застосунку.

**Мета:** 
Розробити клієнтоорієнтований веб-застосунок інтернет-магазину побутової техніки, головним завданням якого є забезпечення максимально зручного, швидкого та прозорого процесу онлайн-шопінгу. Метою системи є створення інтуїтивно зрозумілого середовища, де клієнт зможе:
- легко знаходити потрібну техніку завдяки зручній системі пошуку та фільтрації;
- комфортно переглядати каталог і оформлювати замовлення з будь-якого пристрою (смартфона, планшета чи ПК) завдяки адаптивному дизайну;
- отримувати автоматизований сервіс вищого рівня, що економить час (миттєвий розрахунок умов безкоштовної доставки, автоматична генерація електронних гарантійних талонів тощо).

## Опис предметного середовища

### Актуальність теми
Розвиток електронної комерції робить онлайн-покупки основним способом придбання товарів, зокрема побутової техніки. Необхідність створення зручного та швидкого інтернет-магазину, який коректно відображатиметься на будь-яких пристроях (смартфонах, планшетах, десктопах), є критично важливою для успішного ведення бізнесу.
    
### Об'єкт та предмет роботи
- **Об'єкт:** Веб-застосунок інтернет-магазину побутової техніки.
- **Предмет:** Адаптивний користувацький інтерфейс, структура даних та алгоритми обробки замовлень.

### Бізнес-логіка роботи
- Якщо сума замовлення перевищує 15 000 грн, клієнту автоматично надається безкоштовна доставка.
- Товар не може бути доданий до кошика, якщо його залишок на складі дорівнює нулю.
- До кожної одиниці великої побутової техніки автоматично формується електронний гарантійний талон.

### Вимоги
- **Функціональні:** фільтрація техніки, кошик, оформлення замовлення, розрахунок загальної вартості, email-підтвердження.
- **Нефункціональні:** адаптивність (перебудова для екранів менше 1200px та менше 768px), час завантаження до 2 секунд, шифрування даних (HTTPS).

### Лістинг коду сервера (server.js)

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.use(express.json());

let students = [
    { id: 1, name: "Кириченко Назар", group: "ІС-32" },
    { id: 2, name: "Степаненко Іван", group: "ІС-32" }
];

app.get("/", (req, res) => {
    res.send("Hello from Node.js server");
});

app.get("/students", (req, res) => {
    res.json(students);
});

app.post("/students", (req, res) => {
    const newStudent = req.body;
    const maxId = students.length > 0 ? Math.max(...students.map(s => s.id)) : 0;
    newStudent.id = maxId + 1;
    students.push(newStudent);
    res.status(201).json({ message: "Студента додано успішно", student: newStudent });
});

app.put("/students/:id", (req, res) => {
    const id = parseInt(req.params.id);
    const index = students.findIndex(s => s.id === id);
    if (index !== -1) {
        students[index] = { ...students[index], ...req.body, id: id };
        res.json({ message: "Дані студента оновлено", student: students[index] });
    } else {
        res.status(404).json({ error: "Студента не знайдено" });
    }
});

app.delete("/students/:id", (req, res) => {
    const id = parseInt(req.params.id);
    students = students.filter(s => s.id !== id);
    res.json({ message: `Студента з ID ${id} успішно видалено` });
});

app.listen(port, () => {
    console.log(`Сервер запущено на http://localhost:${port}`);
});
